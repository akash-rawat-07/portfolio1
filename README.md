# my-portfolio

A personal portfolio website built with **React** and **Vite**, styled with **Tailwind CSS**, and enhanced with motion effects.

## Requirements

- **Node.js**: `v22.13.0`
- **npm**: comes with Node.js

## Getting started

Install dependencies:

```bash
npm install
```

Start the dev server:

```bash
npm run dev
```

Then open the URL shown in your terminal (Vite typically uses `http://localhost:5173`).

## Build & preview

Create a production build (outputs to `dist/`):

```bash
npm run build
```

Preview the production build locally:

```bash
npm run preview
```

## Customization

Most of the portfolio content (name, roles, projects, experience, socials, etc.) can be edited in:

- `src/data/portfolio.js`

## Scripts

- `npm run dev` — start dev server
- `npm run build` — build for production
- `npm run preview` — preview the production build
- `npm run lint` — run ESLint
